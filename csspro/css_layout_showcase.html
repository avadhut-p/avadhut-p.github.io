1. Flexbox Showcase (1D Layout)

Flexbox is ideal for distributing space and aligning items in a single dimension (row or column). Observe how items wrap and distribute themselves within the container.

.flex-container {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-around;
    align-items: center;
    gap: 20px;
}
.flex-item {
    flex-grow: 1;
    flex-basis: 150px; /* Minimum width before growing */
}

2. CSS Grid Showcase (2D Layout)

CSS Grid excels at arranging content in both rows and columns simultaneously, giving you precise control over your layout structure.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Creates 3 equal columns */
    gap: 20px;
}

3. Holy Grail Layout (CSS Grid Areas)

This classic layout often features a header, sidebar, main content, and footer. Grid Areas make it incredibly semantic and responsive.

.holy-grail-grid {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header"
        "nav    main"
        "footer footer";
}
/* For smaller screens */
@media (max-width: 768px) {
    .holy-grail-grid {
        grid-template-columns: 1fr;
        grid-template-areas:
            "header"
            "nav"
            "main"
            "footer";
    }
}

4. Repeating Card Layout (Grid with `auto-fit` & `minmax`)

This is extremely powerful for responsive card grids. The cards automatically arrange themselves to fill the available space, wrapping when necessary, without media queries!

.card-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
    gap: 25px;
}
.card img {
    object-fit: cover; /* Ensures image fills container */
    aspect-ratio: 16 / 9; /* Modern way to set aspect ratio */
}

5. Multi-Column Layout (Newspaper Style)

Useful for text-heavy content, this property divides a block of text into a specified number of columns, similar to a newspaper or magazine.

.column-container {
    column-count: 3;
    column-gap: 40px;
    column-rule: 1px solid var(--border-color);
}
.column-container h3 {
    column-span: all; /* Makes sub-heading span all columns */
}

6. Positioning Showcase (`static`, `relative`, `absolute`, `fixed`, `sticky`)

Understanding `position` is crucial for fine-tuning element placement, overlapping, and creating persistent UI elements.

/* Parent must be 'positioned' for absolute children */
#positioning-showcase { position: relative; }

.static-box { position: static; } /* Default */
.relative-box { position: relative; top: 20px; left: 20px; }
.absolute-box { position: absolute; top: 50px; right: 50px; }
.fixed-box { position: fixed; bottom: 20px; right: 20px; } /* Stays in viewport */
.sticky-box { position: sticky; top: 0; } /* Sticks on scroll */

Sticky positioning is incredibly useful for navigation, headers in tables, or sidebars that need to follow the user as they scroll through a specific content area.
